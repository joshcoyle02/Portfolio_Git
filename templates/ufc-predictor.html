About this project

The UFC Predictor is a machine learning-based web application designed to predict the outcome of UFC fights between two selected fighters. Users can choose fighters from a dropdown list, and upon submission, the application utilizes a pre-trained model to forecast the likely winner and the confidence level of that prediction.

This project was devleoped as part of my honours year dissertation, where I explored the idea of using machine learning and data analysis to enhance sports predictions. The model was trained on historical fight data, including fighter statistics, previous match

I trained various machine learning algorithms using two datasets from kaggle that contained historical UFC fight data and fighter statistics. After using a K-fold cross validation to evaluate the models, I found that the Logistic Regression algorithm performed the best with an F1 Score of 0.73.

The confidence metric indicates the probability associated with the predicted winner, as the Logistic Regression model provides probability estimates for each class.
